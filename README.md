# Interactive Nessus Vulnerability Dashboard

Interactive dashboard for vulnerability analysis and reporting based on Nessus scan results.

## 🚀 Features
- **Visual Insights:** Simplified view of critical, high, medium, and low vulnerabilities.
- **Easy Triage:** Designed to help security teams prioritize and manage risks efficiently.
- **Web-Based:** Accessible directly from any modern browser.

## 🛠️ Built With
- HTML5 / CSS3 (Nessus Dashboard Output)
